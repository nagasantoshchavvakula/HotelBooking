This is for Practising Git:)

Git is a distributed version control system designed to track changes in source code during software development:)It will be very Useful now a days in Software Development.It is designed to handle everything from small to very large projects with speed and efficiency.

Git keeps track of changes made to files and directories in a project over time. This allows developers to revert to previous versions, compare changes, and understand the history of the project.This is great! Git enables collaboration through remote repositories, where developers can push their changes and pull updates from others.

GitHub is a web-based platform that provides hosting for Git repositories, allowing you to collaborate on code with others!!!!!!
